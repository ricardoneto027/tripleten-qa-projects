# Sprint 3 — Testes Funcionais Web: Urban Routes

Testes funcionais completos sobre a funcionalidade de **partilha de carros**
do Urban Routes, cobrindo layout, pagamento, reserva e locação de veículos.

---

## 🗺️ Contexto

| Campo | Detalhe |
|---|---|
| Aplicação | Urban Routes — funcionalidade de partilha de carros |
| Tipo de teste | Testes funcionais web |
| Ferramenta | Google Sheets + browser |
| Formato de entrega | Checklists + casos de teste + bug reports |

---

## 📁 Ficheiros

| Ficheiro | Descrição |
|---|---|
| `checklist-layout.md` | Checklist do layout do formulário de reserva |
| `checklist-pagamento.md` | Checklist das janelas "Método de pagamento" e "Adicionar cartão" |
| `test-cases-reserva.md` | Casos de teste para o botão "Reservar" |
| `test-cases-locacao.md` | Casos de teste para a função de locação de carro |
| `bug-reports.md` | Bugs encontrados durante a execução dos testes |

---

## 🔍 Âmbito dos testes

- **Layout**: verificação visual e estrutural do formulário de reserva
- **Pagamento**: fluxo de adição de cartão e seleção de método de pagamento
- **Reserva**: validação do botão "Reservar" — estados, condições e erros
- **Locação**: fluxo completo de locação de veículo
