# Sprint 5 — Mobile Testing: Urban Lunch

Testes de aplicação mobile realizados sobre a app **Urban Lunch**,
com foco na verificação de requisitos funcionais através de emulação.

---

## 📱 Contexto

| Campo | Detalhe |
|---|---|
| Aplicação | Urban Lunch (app de entrega de refeições) |
| Dispositivo emulado | Pixel 5 |
| SO | Android 12.0 ("S") API 31 (via Android Studio) |
| Tipo de teste | Checklist de requisitos funcionais |
| Ferramenta | Android Studio Emulator |

---

## 📁 Ficheiros

| Ficheiro | Descrição |
|---|---|
| `checklist-requisitos.md` | Checklist completa dos requisitos testados |
| `bug-reports.md` | Bugs encontrados durante os testes |
| `screenshots/` | Capturas de ecrã do emulador |

---

## 🔍 Âmbito dos testes

- Navegação e fluxo de encomenda
- Validação de campos e formulários
- Comportamento em diferentes estados de rede
- Verificação visual dos ecrãs principais

---

## ✅ Estado

Projeto submetido e aprovado pela TripleTen.
